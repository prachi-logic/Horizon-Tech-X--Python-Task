# Python Chatbox 🤖

A simple rule-based chatbot built using Python. This chatbot interacts with users through predefined responses and demonstrates the use of Python fundamentals such as functions, conditional statements, loops, and user input handling.

## Features

* Personalized welcome message
* Greeting responses
* Answers common questions
* Joke command
* Simple addition calculator
* Exit command to end the conversation

## Concepts Used

* Functions
* If-Elif Statements
* While Loops
* User Input/Output
* String Methods (`lower()`)

## Sample Commands

* hello
* how are you
* who made you
* favourite colour
* how was your day
* joke
* add
* bye

## How to Run

1. Clone the repository
2. Open the project in Python
3. Run the script:

   ```bash
   python chatbox.py
   ```
